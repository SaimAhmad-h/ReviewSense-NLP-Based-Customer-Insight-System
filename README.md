ReviewSense – NLP-Based Customer Insight System

ReviewSense is an advanced Natural Language Processing (NLP) system designed to extract actionable insights from customer reviews by combining multiple language understanding techniques into a unified pipeline. The system goes beyond basic sentiment detection by incorporating intent classification and topic modeling to provide a deeper understanding of customer feedback.

⚙️ Core Functionality

The system processes raw customer reviews through a multi-stage pipeline:

Text Preprocessing
Cleans and normalizes input text, handling noise, inconsistent formatting, and multilingual variations (especially mixed English and Roman Urdu).
Sentiment Analysis
Determines the emotional tone of the review (positive, negative, neutral), enabling quick assessment of customer satisfaction.
Intent Classification
Identifies the underlying purpose of the review (e.g., complaint, suggestion, praise), allowing more targeted business insights.
Topic Modeling
Extracts key themes and discussion points from large volumes of reviews to highlight recurring issues and trends.
🌍 Multilingual Capability

A key strength of the system is its ability to handle code-mixed language inputs, particularly English and Roman Urdu, making it highly applicable in real-world scenarios where users frequently switch between languages.

🖥️ Real-Time Interface

The project integrates an interactive UI using Gradio, enabling users to:

Input customer reviews in real time
Instantly view sentiment, intent, and topic predictions
Explore insights without needing technical expertise
